## 間違えやすい最新事項(要注意)

1. **登録免許税の免税措置の根拠条文**は、令和8年4月1日から「租税特別措置法第84条の2の3」→**「第84条の2の2」に条数変更**。現行の正しい条文は第84条の2の2。適用期限は**令和9年(2027)3月31日**まで。旧条文を申請書に書くと免税を受けられない恐れがある。詳細は[04](04-tourokumenkyozei.md)。
2. **所有不動産記録証明制度の施行日は令和8年2月2日**(元資料周辺で「2月3日」とする記載は誤り)。詳細は[01](01-seido-houkaisei.md)。
3. **相続登記はオンラインで完結できない**。戸籍が市区町村長の電子署名付き電子文書として発行されないため、申請情報をオンライン送信し戸籍等の書面を別送する「半ライン申請(特例方式)」になる。詳細は[02](02-tetsuzuki-flow.md)。
4. **印鑑証明書の3か月以内制約は相続登記では無い**(遺産分割協議書添付分)。金融機関の相続手続とは扱いが異なる。詳細は[05](05-hitsuyou-shorui-koseki.md)。
