# 02 手続フロー全体像

相続による所有権移転登記を「申請するまで」と「申請してから完了まで」の流れ。

## 1. 相続パターンの判定(最初の分岐)

不動産を誰の名義にするかの根拠は3通り。これにより必要書類と申請書の形が変わる(詳細は[03](03-toukishinseisho.md))。

```
遺言があるか?
├─ あり ─ 特定財産承継遺言(「相続させる」) → 原因「相続」・取得相続人の単独申請
│        遺贈(「遺贈する」)             → 原因「遺贈」(受遺者が相続人か第三者かで分岐)
└─ なし ─ 遺産分割協議をするか?
          ├─ する  → 遺産分割協議による相続(取得者を決める。協議書+印鑑証明書が必要)
          └─ しない → 法定相続分による共同相続(相続人全員の共有。協議書・印鑑証明書は不要)
```

派生論点:
- **数次相続**(相続登記未了の間に相続人が死亡): 中間の相続が単独相続なら1件で直接移転可、中間が複数共有なら別件。
- **代襲相続**(被相続人より先に子等が死亡): 通常の相続として1件で処理。
- **相続放棄**: 放棄者を除いて相続人・相続分を再計算。
- **配偶者居住権の設定**: 別途「配偶者居住権設定」登記(共同申請)。

判定の前提となる相続人の範囲・相続分の算定ロジックは[07](07-souzokunin-soutokubun.md)。

## 2. 申請方式(3方式)

法的効力は同じ。提出経路と電子署名の要否が違う。

| 項目 | (a)書面申請 | (b)オンライン申請 | (c)QRコード付き書面申請 |
|---|---|---|---|
| 申請書作成 | 紙 | 申請用総合ソフト | 申請用総合ソフト |
| 送信 | なし | あり(電子署名付き) | あり(電子署名なし) |
| 電子証明書 | 不要 | 必要 | 不要 |
| ICカードリーダライタ | 不要 | 必要 | 不要 |
| 添付書面 | 紙を持参/郵送 | 原則電子(相続は紙併用) | 紙を持参/郵送 |
| 登録免許税納付 | 収入印紙等 | 電子納付/印紙 | 電子納付/印紙 |

- **(a)書面申請**: すべて紙。窓口持参または郵送。PC環境も電子証明書も不要で誰でもできる。
- **(b)オンライン申請**: 申請用総合ソフトで作成し電子署名を付して送信。自宅から申請でき処理が早い。マイナンバーカードの署名用電子証明書とICカードリーダライタ、ソフト導入が必要。
- **(c)QRコード付き書面申請(二次元バーコード付き書面申請)**: 2020年1月14日開始。ソフトで作成・送信した上でQR付き申請書を印刷し、添付書面とともに窓口持参/郵送する。**電子署名・電子証明書が不要**でミス低減・処理高速化・電子納付が使える「いいとこ取り」。最終的に紙を提出する点は残る。

相談予約の後、書面での窓口申請を推奨

### 公式システム
- **登記供託オンライン申請システム(登記ねっと供託ねっと)**: 法務省の公式システム総称。中に「申請用総合ソフト」と「かんたん登記供託申請」がある。
- **申請用総合ソフト**: 全手続対応のPCインストール型無償ソフト。作成・電子署名・送信・電子公文書取得まで一貫。相続登記の(b)(c)いずれにも使う。
- **かんたん登記供託申請(旧かんたん証明書請求)**: ブラウザだけで使える簡易版。証明書請求や相続人申告登記の申出はできるが、**相続による所有権移転登記(本登記)は対象外**。

## 3. なぜオンラインで完結できないか(半ライン申請)

- 完全オンラインでは添付情報も作成者の電子署名付き電子文書が原則。だが相続登記に必須の戸籍全部事項証明書・除籍謄本・改製原戸籍は、市区町村長の電子署名付き電子文書として発行されていない。よって相続登記はオンラインのみで完結できない。
- **特例方式(半ライン申請)**: 申請情報をオンライン送信し、紙の添付書面(戸籍・遺産分割協議書・印鑑証明書等)を後日、管轄法務局へ郵送/持参する。相続関係説明図を添付すると戸籍原本の還付を受けやすい。
- 要確認: 紙の添付書面の到着期限(「受付後2日以内」とする民間解説が多いが一次情報での日数明示は未確認、管轄に要確認)。

## 4. 管轄・期間・補正・完了書類

- **管轄法務局**: 不動産所在地を管轄する登記所。法務局「管轄のご案内」から検索。
- **処理期間**: 各法務局が「登記完了予定日」を公表。おおむね1〜2週間が目安。都市部・年度末は3週間以上、地方・閑散期は5日前後のことも。オンライン/QR申請は書面より数日早い傾向。
- **補正**: 不備があると登記官から補正を求められ、応じれば取り下げずに修正できる。連絡は電話等が一般的。オンライン申請の補正・取下げはオンラインで行う。
- **完了書類**: 登記完了証と登記識別情報通知が交付される。オンライン申請では申請用総合ソフトの処理状況表示から電子公文書としてダウンロード可。書面交付希望は窓口受取または郵送(本人限定受取郵便等)。返送を郵送で希望する場合は返信用封筒・切手を添付。

## 5. 費用の全体像

3本柱: 登録免許税+戸籍等取得実費+(依頼するなら)司法書士報酬。

- **登録免許税**: 相続による所有権移転は固定資産税評価額の0.4%(1000分の4)。計算詳細は[04](04-tourokumenkyozei.md)。
- **戸籍等取得実費**: 1通数百円〜750円程度。出生から死亡まで揃えるため通数がかさみ総額数千円〜1万円超。
- **司法書士報酬**: 自由化されており事務所差が大きいが、おおむね6万〜15万円(中心7万〜10万円)が民間集計の目安(要確認)。

## 6. 提出前チェックリスト(元資料のチェックリストを再構成)

申請書記載事項:
- [ ] 登記の目的は正しいか(持分移転なら「○○持分全部移転」、混在なら併記)
- [ ] 不動産(土地・建物)の表示は登記事項証明書と一致しているか
- [ ] 申請する建物は登記されているか(未登記建物は登記申請できない)
- [ ] 区分建物(マンション)なら敷地権の表示も書いたか
- [ ] 課税価格は「価格(評価額)」で計算したか(課税標準額ではない)
- [ ] 登録免許税は正しい金額か(相続=課税価格の1000分の4、100円未満切捨)
- [ ] 申請書1枚目から登録免許税貼付台紙までの契印(割印)をしたか
- [ ] 連絡先の電話番号(昼間に連絡が付くもの)を書いたか

添付書類:
- [ ] (代理時)委任状を作成し日付・委任事項・不動産の表示を記載したか
- [ ] 申請書記載の添付情報がすべて揃っているか
- [ ] 評価額が確認できる書類(評価証明書・名寄帳・課税明細)があるか

相続関係:
- [ ] 相続証明情報(被相続人の出生〜死亡の戸除籍)は全部そろっているか(法定相続情報一覧図取得済みなら戸除籍添付不要)
- [ ] 登記簿上の所有者の住所と被相続人の戸籍の本籍は同じか(違えば同一性証明が必要)
- [ ] 相続人全員の現在戸籍(被相続人の死亡後取得)があるか
- [ ] 相続する人の住民票または戸籍の附票があるか
- [ ] (遺産分割時)協議書に相続人の実印押印と印鑑証明書の添付があるか
- [ ] (遺言時)遺言の内容と登記申請内容は一致しているか

その他:
- [ ] 返送用の封筒・切手を準備したか(本人限定受取郵便)
- [ ] 申請先の法務局(不動産所在地の管轄)は正しいか
