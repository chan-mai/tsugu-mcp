## 1. 申請書各項目の記載ルール

### 1-1. 登記の目的
- 原則: `所有権移転`
- 被相続人が持分のみ所有: `(被相続人氏名)持分全部移転`(例 `法務太郎持分全部移転`)。この場合は申請人欄で移転持分も記載。
- 混在(複数不動産で全部移転と持分移転が併存): `所有権移転及び(被相続人氏名)持分全部移転`

### 1-2. 原因
- 書式: `令和○年○月○日相続`
- 日付は被相続人の戸籍上の**死亡日**。遺産分割の場合も協議成立日ではなく死亡日を書く。
- 数次相続では複数原因を連記(後述2-E)。

### 1-3. 相続人/申請人(法定相続の公式記載例)
```
相 続 人　　（被相続人　法務太郎）
　（申請人）　○○市○○町二丁目１２番地
　　　　　　　持分２分の１　法務花子　印
　　　　　　　　氏名ふりがな　　ほうむ　はなこ
　　　　　　　　生年月日　　　　昭和３０年４月５日
　　　　　　　　メールアドレス　abcdefg123@example.com
　　　　　　　○○郡○○町○○３４番地（住民票コード１２３４５６７８９０１）
　　　　　　　　持分４分の１　法務一郎
　　　　　　　○○市○○町三丁目４５番６号
　　　　　　　　持分４分の１　法務貴子
　　　　　　　連絡先の電話番号００－００００－００００
```
- 被相続人は `(被相続人 ○○○○)` と冠記。
- 申請人は 住所→持分→氏名 の順。氏名末尾に押印(認印で可)。`(申請人)` は申請人になる者に付す。
- ふりがな・生年月日・メールアドレスは**申請人である相続人のみ**記載(申請人でない者・海外居住者は不要)。メールは本人専用のもの。
- 外国人: `ジョン・スミス（ＪＯＨＮ ＳＭＩＴＨ）` とローマ字併記、ふりがな不要。住所が海外なら国内連絡先となる者の氏名・住所も記載。
- 連絡先電話番号: 補正連絡用、平日日中連絡可能なもの。
- **住民票コード**を住所の後に `（住民票コード１２３４５６７８９０１）` と書くと住所証明情報(住民票)を省略可。

### 1-4. 持分の書き方(共有=複数申請人)
- `持分○分の○` を各人の住所の次行・氏名の前に置く。
- 法定相続分の早見・算定ロジックは[07](07-souzokunin-soutokubun.md)。

### 1-5. 添付情報
書式は `添付情報` の次行に `登記原因証明情報　住所証明情報` と列挙。代理時は `代理権限証明情報` を追加。
- 登記原因証明情報: ケースで中身が変わる(後述2)。詳細は[05](05-hitsuyou-shorui-koseki.md)。
- 住所証明情報: 取得相続人の住民票の写し(マイナンバー不記載)。住民票コード記載で省略可。
- 法定相続情報一覧図の写しを提出するか法定相続情報番号を記載すれば、被相続人の戸除籍と相続人の戸籍の添付に代替可。一覧図に相続人の現住所が記載されていれば住所証明にも代替可。

### 1-6. 登記識別情報の通知
- 書式: `□登記識別情報の通知を希望しません。` 希望しない場合に□チェック。
- 共同相続で申請人にならない相続人には登記識別情報は通知されない。

### 1-7. 申請日・管轄
- 書式: `令和○年○月○日申請　○○法務局（又は地方法務局）○○支局（又は出張所）`
- 管轄は不動産所在地を管轄する法務局。

### 1-8. 課税価格・登録免許税
- 書式: `課税価格　金２，０００万円` / `登録免許税　金８０，０００円`
- 相続の税率は1000分の4。端数処理・免税の記載は[04](04-tourokumenkyozei.md)。
- 免税の場合は課税価格・税額を書かず、登録免許税欄に根拠条項を記載。

### 1-9. 不動産の表示
- 土地: 所在・地番・地目・地積(例 `１２３・４５平方メートル`、小数点は全角中黒)。
- 建物: 所在・家屋番号・種類・構造・床面積。
- **不動産番号(13桁)** を記載すると土地の所在・地番・地目・地積(建物は所在・家屋番号・種類・構造・床面積)の記載を省略できる。
- 登記事項証明書どおりに正確に転記すること。

## 2. ケース別パターン

各ケースで「登記の目的・原因・申請構造・添付情報・税率」がどう変わるかを示す。一覧サマリーは末尾の表。

### (A) 法定相続分による相続(共同相続、遺産分割なし)
| 項目 | 内容 |
|---|---|
| 登記の目的 | 所有権移転 |
| 原因 | `令和○年○月○日相続`(死亡日) |
| 申請構造 | 相続人全員に法定相続分の持分を記載。相続人1人が保存行為として全員のため申請可(委任状不要、ただし申請人にならない者には識別情報が通知されない) |
| 添付情報 | 登記原因証明情報(被相続人の出生〜死亡の戸除籍+相続人全員の戸籍)、住所証明情報(相続人全員の住民票)。**遺産分割協議書・印鑑証明書は不要** |
| 税率 | 4/1000 |

### (B) 遺産分割協議による相続(特定の相続人が取得)
公式記載例(子1人が単独相続):
```
登記の目的　　　所有権移転
原　　　　因　　令和７年６月２０日相続
相　続　人　　（被相続人　法務太郎）
　　　　　　　　○○郡○○町○○３４番地（住民票コード１２３４５６７８９０１）
　　　　　　　　法務一郎　印
```
| 項目 | Aとの差分 |
|---|---|
| 目的・原因 | 同じ(原因日付は死亡日) |
| 申請構造 | 取得する相続人のみ。**単独取得なら持分記載不要**(氏名のみ)。共有取得なら各人に持分。取得しない相続人は記載しない |
| 添付情報 | 登記原因証明情報=戸除籍一式+**遺産分割協議書**。協議書には協議をした相続人全員の**印鑑証明書(実印の証明)**が必須。住所証明は取得者のみ |
| 税率 | 4/1000 |

遺産分割協議書の作り方は[06](06-fuzui-shorui.md)。

### (C) 遺言による相続 — 特定財産承継遺言(「相続させる」遺言)
| 項目 | 内容 |
|---|---|
| 登記の目的 | 所有権移転 |
| 原因 | `令和○年○月○日相続`(「遺贈」ではなく「相続」。死亡日) |
| 申請構造 | 当該相続人の**単独申請** |
| 添付情報 | 登記原因証明情報=遺言書+被相続人の死亡記載の戸除籍+相続人の戸籍(死亡日以後の証明日)。住所証明=取得相続人の住民票 |
| 税率 | 4/1000 |

遺言書の種別:
- 公正証書遺言: 検認不要。
- 自筆証書遺言: 原則、家庭裁判所の検認済証明書付きが必要。ただし遺言書保管法で法務局保管の自筆証書遺言は検認不要(遺言書情報証明書を用いる)。

A(法定相続)/B(遺産分割)/C(特定財産承継遺言)はいずれも目的「所有権移転」・原因「相続」・税率4/1000が共通。差は申請人の範囲と登記原因証明情報の中身(戸籍のみ/協議書+印鑑証明書/遺言書)。

### (D) 遺贈(「遺贈する」遺言)
受遺者が相続人か第三者かで申請構造・税率が分岐。

**D-1 相続人に対する遺贈(令和5年4月1日施行の単独申請)**
公式記載例:
```
登記の目的　所有権移転
原　　因　　令和４年８月２０日遺贈
権 利 者　　○○市○○町一丁目２３番４号
(申請人)　　（住民票コード１２３４５６７８９０１）
　　　　　　法務花子　印
義 務 者　　○○市○○町五丁目６７番８号
　　　　　　法務太郎
```
| 項目 | 内容 |
|---|---|
| 登記の目的 | 所有権移転 |
| 原因 | `令和○年○月○日遺贈`(遺贈者の死亡日) |
| 申請構造 | 受遺者である相続人の**単独申請**。様式上は権利者・義務者を併記するが義務者は申請人にならない。義務者の押印・印鑑証明書・登記識別情報の添付は不要 |
| 添付情報 | 登記原因証明情報(遺言書、遺贈者の死亡記載の戸除籍、受遺者が相続人と分かる戸籍)、住所証明情報(受遺者の住民票) |
| 税率 | 4/1000 |

**D-2 第三者(相続人以外)への遺贈** — 共同申請(実務上確立、公式専用様式なし=要確認)
| 項目 | 内容 |
|---|---|
| 登記の目的 | 所有権移転 |
| 原因 | `令和○年○月○日遺贈` |
| 申請構造 | **共同申請**。権利者=受遺者(第三者)、義務者=遺言執行者がいればその者、いなければ相続人全員 |
| 添付情報 | 遺言書、遺贈者の死亡記載の戸除籍、義務者の**登記識別情報(権利証)**、義務者の**印鑑証明書**(3か月以内)、遺言執行者の資格証明(遺言指定なら遺言書、家裁選任なら選任審判書)、受遺者の住所証明 |
| 税率 | **20/1000** |

税率の根拠: 相続・相続人への遺贈は1000分の4、第三者への遺贈は1000分の20(登録免許税法別表第一)。

### (E) 数次相続
登記名義人の死亡後、その登記が未了の間に相続人が死亡して次の相続が開始したケース。

中間省略の可否(公式記載例の明文): 「最終の相続以外の相続について、共同相続人数名が権利を取得している場合には、一件の申請では登記することができません」
- 中間の相続が**単独相続**(中間相続人が1人。または放棄・遺産分割で結果1人になる) → 1件で登記名義人から最終相続人へ直接移転可。
- 中間が**複数共有** → 相続ごとに別件申請。

公式記載例(太郎死亡→中間=春夫が単独相続→春夫死亡→最終=一郎・温子):
```
登記の目的　所有権移転
原　　因　　平成２年３月２１日法務春夫相続
　　　　　　令和７年６月２０日相続
相 続 人　　（被相続人　法務太郎）
　　　　　　持分２分の１　法務一郎
　　　　　　持分２分の１　法務温子
```
原因連記の構造:
- 1行目 `平成2年3月21日法務春夫相続` = 第1被相続人(太郎)の死亡日+中間相続人(春夫)の氏名+「相続」。
- 2行目 `令和7年6月20日相続` = 第2相続(春夫)の死亡日+「相続」。
- 被相続人欄は登記名義人である太郎(中間の春夫ではない)。中間相続人の氏名は原因欄にのみ登場。

添付情報: 第1被相続人の出生〜死亡の戸除籍+中間・最終相続人全員の戸籍(相続の連鎖が戸籍上たどれること)、最終相続人の住所証明。遺産分割を伴えば協議書。

### (F) 代襲相続
被相続人の死亡時点で既に相続人(子)が死亡しており孫が相続。**法律上1回の相続**として扱い、通常の相続様式をそのまま使う(専用様式なし)。
| 項目 | 内容 |
|---|---|
| 登記の目的 | 所有権移転 |
| 原因 | `令和○年○月○日相続`(被相続人の死亡日、**単一日付**) |
| 申請構造 | 代襲相続人(孫)を相続人として記載、取得持分を併記。申請書に「代襲」の特別注記は不要 |
| 添付情報 | 通常の相続戸籍に加え**被代襲者(先に死亡した子)の出生〜死亡の連続戸除籍**が必須 |

数次相続との違い: 代襲相続は子が被相続人より**先に**死亡(相続1回、原因日付は単一)。数次相続は被相続人の死亡**後**に相続人が死亡(相続2回以上、原因日付を連記、中間相続人の配偶者も相続人になりうる)。判定ロジックは[07](07-souzokunin-soutokubun.md)。

### (G) 相続放棄があった場合
放棄者は初めから相続人でなかったものとみなされる(民法939条)。
| 項目 | 内容 |
|---|---|
| 登記の目的・原因 | 通常の相続と同じ |
| 申請構造 | 放棄者は申請書に一切記載しない。放棄者を除いた相続人で法定相続分を**再計算** |
| 添付情報 | 放棄の事実は戸籍に記録されないため、家裁発行の**相続放棄申述受理証明書**を登記原因証明情報の一部として追加(被相続人の氏名・死亡日・本籍記載の受理通知書でも可とされるが、本籍記載のない通知書は不可の可能性が高い) |
| 税率 | 4/1000 |

同順位全員が放棄すると次順位(直系尊属→兄弟姉妹)へ相続権が移り、その者の戸籍も必要になる。

### (H) 配偶者居住権の設定登記
民法改正で令和2年4月1日施行。相続による所有権移転とは別の登記。
| 項目 | 内容 |
|---|---|
| 登記の目的 | `配偶者居住権設定` |
| 申請構造 | **共同申請**。権利者=配偶者、義務者=居住建物を取得した相続人 |
| 原因 | 取得の由来を書く: 遺産分割→`令和○年○月○日遺産分割`、遺贈→`令和○年○月○日遺贈`、死因贈与→`令和○年○月○日贈与`(「設定」とは書かない) |
| 登記事項 | **存続期間**は必須(別段の定めなければ `配偶者居住権者の死亡時まで`)。第三者に使用収益させる定めがあれば併記 |
| 添付情報 | 登記原因証明情報、義務者(建物所有者)の登記識別情報・印鑑証明書、固定資産評価証明書 |
| 税率 | **2/1000**(建物の評価額) |
| 対象 | **建物にのみ**設定(土地不可)。前提として建物の相続登記が必要(連件同時申請可) |

## 3. 区分建物(マンション)の不動産の表示

不動産の表示の書式は登記の目的によらず共通。3ブロック構成。
```
不動産の表示
　不動産番号　　　　　１２３４５６７８９０１２３
　一棟の建物の表示
　　所　　　　在　　　○○市○○町一丁目２３番地
　　建物の名称　　　　○○マンション
　専有部分の建物の表示
　　家 屋 番 号　　　　○○町一丁目２３番の３０１
　　建物の名称　　　　３０１号
　　種　　　　類　　　居宅
　　構　　　　造　　　鉄骨造１階建
　　床 面 積　　　　　３階部分　６０・１２平方メートル
　敷地権の表示
　　符　　　　号　　　１
　　所在及び地番　　　○○市○○町一丁目２３番
　　地　　　　目　　　宅地
　　地　　　　積　　　５００・００平方メートル
　　敷地権の種類　　　所有権
　　敷地権の割合　　　１０００分の３５
```
- 内訳: 一棟の建物の表示(所在・建物の名称)/専有部分の建物の表示(家屋番号・建物の名称・種類・構造・床面積)/敷地権の表示(符号・所在及び地番・地目・地積・敷地権の種類・敷地権の割合)。
- 不動産番号で省略する場合でも、区分建物は**「敷地権の種類」「敷地権の割合」は省略不可**。

## 4. 物理的作成・綴じ方・提出

### 用紙・文字
- A4・横書き・上質紙等(長期保存可能な丈夫な紙)、左とじ。
- パソコン入力か黒色インク・黒色ボールペン等。消せるインク・鉛筆は不可。

### 契印(複数枚の場合)
- 申請人(2人以上なら**そのうち1人で可**)が、ホチキスどめした各用紙のつづり目に契印。

### 綴じ方・順番
- 上から「登記申請書」→「収入印紙等貼付台紙」を左余白2か所ホチキスどめ。
- 添付書類(戸籍・遺産分割協議書等)はその後ろに**クリップどめ**(ホチキスで一体化しない)、全体は左とじ。

### 収入印紙貼付台紙
- 収入印紙は申請書に直接貼らず別の白紙(台紙)に貼り、申請書とつづって申請書と台紙のつづり目に契印。**収入印紙そのものには押印しない**(消印は登記所が行う)。免税なら台紙不要。

### 文字の訂正(不動産登記規則45条2項)
- 字数を欄外記載するか、該当文字に括弧等の記号を付して範囲を明示し当該部分に押印(訂正印)。元の文字は塗りつぶさず判読可能に残す。

### 提出方法
- 申請先は不動産所在地を管轄する法務局。窓口持参・郵送(「不動産登記申請書在中」+書留、配達日=申請日)・オンライン・QRコード付き書面申請([02](02-tetsuzuki-flow.md))。
