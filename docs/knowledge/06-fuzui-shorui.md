# 06 付随書類(協議書・委任状・相続関係説明図)

遺産分割協議書・委任状・相続関係説明図・法定相続情報一覧図の記載要件。公式記載例の原文は全角。

---

## 1. 遺産分割協議書

### 1-1. 公式記載例(登記手続ハンドブック)
```
　　　　　　遺 産 分 割 協 議 書

令和2年3月20日、〇〇市〇〇町〇番地 法務太郎 の死亡によって開始した相続の
共同相続人である法務花子、法務一郎及び法務温子は、本日、その相続財産について、
次のとおり遺産分割の協議を行った。

相続財産のうち、下記の不動産は、法務一郎(持分2分の1)及び法務温子(持分2分の1)
が相続する。

この協議を証するため、本協議書を3通作成して、それぞれに署名、押印し、
各自1通を保有するものとする。

令和2年7月1日
　〇〇市〇〇町二丁目12番地　　法務花子 実印
```

### 1-2. 必須記載事項
- 相続開始日(被相続人の死亡年月日)+被相続人の特定(住所/地+氏名)
- 共同相続人全員の列挙
- 取得者と取得財産の対応(誰がどの財産を取得するか。持分があれば持分も)
- 作成年月日(協議成立日。相続開始日とは別)
- 相続人全員の住所+氏名+実印押印

公式の協議書本文は被相続人を「死亡日+住所+氏名」で特定するにとどまる。本籍・最後の住所・登記簿上の住所は添付書類(戸籍・住民票除票・戸籍附票)で証明する建付け。実務慣行として被相続人欄に「氏名・生年月日・死亡日・最後の本籍・最後の住所(登記簿上の住所)」をまとめ書きし同一性を示すことが多い(公式必須ではない)。

### 1-3. 不動産の表示
登記事項証明書どおりに正確記載する。地番・家屋番号で特定し住居表示(住所)と混同しない。土地・建物・区分建物の書式は[03](03-toukishinseisho.md)と同じ。協議書は当事者特定書面なので不動産番号で省略せず登記事項どおりフル記載が安全(実務助言)。

### 1-4. 預貯金・有価証券・包括条項(公式例になし=実務文例)
- 預貯金: `◯◯銀行◯◯支店 普通預金 口座番号0000000`(金融機関名・支店・種別・口座番号で特定)
- 株式: `◯◯株式会社 普通株式 000株`
- 財産多数時: 「別紙財産目録記載のとおり各相続人が取得する」と本文で受け目録で指定
- 包括条項(後日判明財産の帰属):
  - 取得者指定型: 「本協議書に記載のない遺産が後日判明した場合は、当該遺産は○○が取得するものとする。」
  - 法定相続分型: 「…各相続人の法定相続分の割合で取得するものとする。」
  - 再協議型: 「…相続人全員で改めて協議する。」

### 1-5. 署名・実印・印鑑証明書
- 相続人全員が実印押印+印鑑証明書を各1通添付。
- 相続登記では印鑑証明書に**作成後3か月以内の制約なし**(発行から3か月超でも使える)。これは相続登記限定で、金融機関の相続手続では3〜6か月以内を求められるのが通例。
- 登記申請人(=登記名義人になる相続人)本人の印鑑証明書は登記添付としては不要(ただし全員実印押印は必要)。

### 1-6. 遺産分割協議書 vs 遺産分割協議証明書
| 項目 | 遺産分割協議書 | 遺産分割協議証明書 |
|---|---|---|
| 形式 | 1通に全員が署名押印(契約書型) | 相続人ごとに個別書面で自己の取得分を証明(証明書型) |
| 署名押印 | 全員が同一書面 | 各人が別々の書面、全員分を揃える |
| 効力 | 同等 | 同等 |
| 向くケース | 相続人が集まれる | 遠隔地・海外・疎遠で持ち回りが困難 |

協議証明書は同じ雛形を全員へ同時送付でき各自返送で済むため迅速(相続人4人なら4通)。

### 1-7. 数次相続・代襲相続の当事者表示
- 数次相続: 後で死亡した中間者を「相続人兼被相続人 ○○(令和○年○月○日死亡)」と表示し、署名欄は「相続人兼 ○○の相続人 [住所] ●● 実印」。一次・二次の共同相続人が同一なら1通にまとめる実務。
- 代襲相続: 代襲相続人も「相続人」として署名・実印押印。「代襲相続人」とは書かないのが基本(関係明示のため「(被代襲者 ○○)」を併記する書式もある)。「相続人兼被相続人」表記になるのは数次相続のみ。

### 1-8. 未成年者・成年被後見人
- 未成年者: 親権者も共同相続人なら利益相反(民法826条1項)→子のため家庭裁判所で特別代理人を選任(申立てに協議書案を添付)。協議書には「未成年者○○ 特別代理人 △△」と署名し特別代理人の実印+印鑑証明書。
- 成年被後見人: 成年後見人が代理参加・署名押印。後見人自身も同じ相続の共同相続人なら利益相反→後見監督人がいれば監督人、いなければ家裁に特別代理人選任を申立て。

### 1-9. よくある不備
- 訂正: 二重線→正しい文字→押印、欄外に「○行目 ○字削除○字加入」。全員に関わる訂正は全員の訂正印(実印)。
- 契印: 複数枚ならページ綴じ目に相続人全員が実印で契印(製本テープ製本時は表紙と本紙にまたがる契印)。
- 捨印: 軽微な誤字脱字の訂正に備えるが、相続人変更・取得者変更など実質変更には使えない。悪用リスクがあり付すかは慎重に。
- 住所不一致: 協議書の住所は印鑑証明書の住所と一致させる(「番地/番」「丁目」の表記揺れも不一致扱いになり得る)。引越しで現住所と異なっても協議の効力は失われず、住民票除票・戸籍附票で連続性を示せば登記可能。
- 相続人漏れ: 相続人全員の参加・合意が必須。1人でも欠けると協議は無効。所在不明者は不在者財産管理人選任や遺産分割調停で全員関与の形を整える。

## 2. 委任状(代理人申請)

### 2-1. 公式記載例
```
　　　　　委　　任　　状
　私は、○○市○○町○○番地　乙野二郎　に、次の権限を委任します。

１　下記の登記に関し、登記申請書を作成すること及び当該登記の申請に必要な書面と共に登記申請書を管轄登記所に提出すること
２　登記が完了した後に通知される登記識別情報通知書及び登記完了証を受領すること
３　登記の申請に不備がある場合に、当該登記の申請を取下げ、又は補正すること
４　登記に係る登録免許税の還付金を受領すること
５　上記１から４までのほか、下記の登記の申請に関し必要な一切の権限

　令和７年７月１日
○○市○○町二丁目１２番地
法　務　花　子　　印 ← 認印で結構

　　　　　　　　　　　記
登記の目的　　所有権移転
原　　　因　　令和７年６月２０日相続
相　続　人　　(被相続人　法務太郎)
○○市○○町二丁目１２番地　　持分２分の１　法務花子
…
不動産の表示
所　在　○○市○○町一丁目 / 地　番　２３番 / 地　目　宅地 / 地　積　１２３・４５平方メートル
```

### 2-2. ポイント
- 委任者(本人=相続人)の住所・氏名+押印。相続登記の委任状は**認印で可**(公式注記に明記)。遺産分割協議書(全員実印+印鑑証明)とは要求水準が異なる。
- 委任者の住所は住民票上の住所。
- 委任事項は1〜5号(申請書の作成・提出/識別情報通知書・完了証の受領/補正・取下げ/還付金受領/包括条項)。法務局には2つの公式版があり最終版は4号が還付金受領、もう1版(様式2)は4号が複代理人選任。安全側に倒すなら必要な権限を統合し最終号の包括条項「その他必要な一切の権限」を必ず置く。「原本還付請求・受領」を明示する条項はどちらの公式版にもなく民間テンプレートが追加する実務慣行。
- 不動産多数時は「不動産の表示は別紙目録のとおり」とし目録を別紙添付(複数枚は契印)。
- 委任状自体は原本還付の対象外。

### 2-3. 委任状が要るとき/要らないとき
- 委任状が要るのは「申請人本人でない者が代理する」とき。
- 相続人1人が保存行為として共同相続人全員のため法定相続登記を申請する場合、他相続人の委任状は不要(その相続人自身が申請人)。ただし申請人にならなかった相続人には登記識別情報が通知されないため、代表者が全員分の識別情報を受け取るには他相続人からの委任状(識別情報受領の委任)を取る運用が推奨。
- 遺産分割で1人が単独取得し本人が申請するなら委任状不要、第三者委任時のみ必要。

## 3. 相続関係説明図

### 3-1. 役割(原本還付)
「相続関係説明図」を提出すると、申請書に添付した登記原因証明情報として提出した戸籍全部(個人)事項証明書・除籍事項証明書を、登記の調査終了後に返してもらえる(原本還付の手続)。通常の原本還付はコピー添付が必要だが、戸籍はこの図を出せばコピーなしで原本を返してもらえる。ただし遺産分割協議書は相続関係説明図では原本還付できず別途その謄本(コピー)が必要。

### 3-2. 記載事項(公式最小例)
公式記載例の見出し:
- タイトル: 「被相続人 法務太郎 相続関係説明図」
- 被相続人: 住所/死亡(年月日)/(被相続人)/氏名 ← 本籍・出生は記載なし
- 各相続人: 住所/出生(年月日)/(相続人)/氏名

実務標準(公式必須ではない): 被相続人に「最後の本籍」「出生年月日」を追加し、続柄(妻・長男等)を併記。相続関係説明図は様式自由で、登記官が相続関係を確認できれば足りる。

### 3-3. 注記の意味
公式が使う注記は「(相続人)」「(分割)」の2つ。
| 注記 | 意味 | 区分 |
|---|---|---|
| (相続人) / (相続) | 当該不動産を相続する人 | (相続人)=公式、(相続)=実務 |
| (分割) | 相続人だが遺産分割でその不動産を取得しない人 | 公式 |
| (放棄) | 家裁で相続放棄した人(民法939条で初めから相続人でないとみなす) | 実務慣行(公式例になし) |

その他、申出人/申請人に「(申出人)」、代襲で先死亡の被代襲者に「亡」+死亡年月日を付す実務。

### 3-4. 図の描き方
- 線の慣行(法令ルールではない): 配偶者と被相続人=二重線、親子=一本線。
- 数次相続: 1枚に第1被相続人と中間で死亡した相続人を図示し、登記原因も2段書き。
- 代襲相続: 被代襲者(先死亡の子)に「亡」+死亡年月日、その下に代襲者を一本線でつなぐ。
- 兄弟姉妹相続(第3順位): 被相続人の父母を上に置き「亡」注記、その下に被相続人と兄弟姉妹を並列配置。半血(異父母)兄弟姉妹は相続分が全血の1/2のため続柄を区別。

## 4. 相続関係説明図 vs 法定相続情報一覧図

| 観点 | 相続関係説明図 | 法定相続情報一覧図の写し |
|---|---|---|
| 性質 | 私文書(作成者が作る図、公的効力なし) | 登記官の認証文付き写し(公的証明書) |
| 認証 | なし | あり |
| 戸籍 | 戸籍束と一緒に提出→原本還付で束が返る | 戸籍束の代わりに1枚で各種手続に使える |
| 住所記載 | 実務上記載するのが通常 | 任意。記載すれば後続手続で住民票添付を省略可 |
| 様式自由度 | 完全自由 | 図形式原則(列挙形式も可)、A4・下部余白等の制約あり |
| 用途 | 主に相続登記の原本還付 | 相続登記+預貯金払戻し+相続税申告+年金等を横断利用 |
| 手数料・制度 | — | 交付無料、平成29年5月29日運用開始 |
| 数次相続 | 1枚に複数の被相続人をまとめて図示可 | 被相続人1人につき1枚、被相続人ごとに別々に作成 |

法定相続情報一覧図の認証文(末尾): 「これは、…当局保管に係る法定相続情報一覧図の写しである。」+登記官名・職印。注記で「相続放棄に関しては本書面に記載されない」「相続手続以外に利用できない」とされる。続柄は戸籍記載のほか申出人の選択で「子」とも記載可(ただし「子」と記載すると相続税申告等で使えない手続がある)。

法務局は相続パターン別に一覧図の様式(Excel)と記載例を公開(配偶者・子/子のみ/配偶者・親/配偶者・兄弟姉妹/異父母兄弟姉妹/代襲/再代襲/養子含む/列挙形式など)。
