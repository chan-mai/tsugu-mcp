# 01 制度と法改正

所有者不明土地問題の解消を目的とする一連の改正(民法等の一部を改正する法律=令和3年法律第24号、相続土地国庫帰属法=令和3年法律第25号)を中心に、相続登記まわりの制度を整理する。2026-06-27時点で下表はすべて施行済み。

| 施行日 | 制度 | 根拠 |
|---|---|---|
| 令和5年(2023)4月27日 | 相続土地国庫帰属制度 | 相続土地国庫帰属法 |
| 令和6年(2024)3月1日 | 戸籍の広域交付 | 戸籍法改正(令和元年法律17号) |
| 令和6年(2024)4月1日 | 相続登記の申請義務化/相続人申告登記 | 不登法76条の2・76条の3 |
| 令和7年(2025)4月21日 | 検索用情報の申出の受付開始 | 不登法76条の6関連 |
| 令和8年(2026)2月2日 | 所有不動産記録証明制度 | 不登法119条の2 |
| 令和8年(2026)4月1日 | 氏名・住所等変更登記の義務化/職権変更登記 | 不登法76条の5・76条の6 |

## 1. 相続登記の申請義務化(令和6年4月1日施行)

### 根拠条文
- 申請義務: 不動産登記法第76条の2第1項
- 遺産分割成立時の追加義務: 同第76条の2第2項
- 過料: 同法第164条第1項

### 申請期限(2段階)
1. **基本の義務**: 「自己のために相続の開始があったことを知り、**かつ**その不動産の所有権を取得したことを知った日から3年以内」に相続登記を申請。
2. **遺産分割成立時の追加義務**: 遺産分割が成立して不動産を取得した相続人は、**遺産分割が成立した日から3年以内**に、その内容を踏まえた登記を申請。相続人申告登記をしていても、後に遺産分割が成立すれば改めてこの3年義務が走る。

### 施行日前に開始した相続の経過措置
- 令和6年4月1日**より前**に開始した相続も義務化の対象。
- 期限は「施行日(令和6年4月1日)」と「自己のために相続開始・所有権取得を知った日」の**いずれか遅い日から3年以内**。
- 実務上の表現: 施行前にすでに相続・取得を知っていたケースは**令和9年(2027)3月31日まで**(民法等一部改正法附則第5条第6項)。

### 正当な理由(過料を免れる類型・公式列挙)
1. 相続人が極めて多数で、戸籍収集等に多くの時間を要する
2. 遺言の有効性や遺産の範囲等が争われている
3. 申請義務者本人が重病等
4. 申請義務者がDV被害者等で生命・身体に危害のおそれ
5. 申請義務者が経済的に困窮して費用負担が困難

### 過料(10万円以下)の運用
- 自動賦課ではない。流れは「登記官が義務違反を職務上知る → 相当の期間を定めて申請を**催告** → 期限内に申請がなければ裁判所に**通知** → 裁判所が過料を科すか判断」。
- 催告を受けた相続人の説明により、登記官が正当な理由ありと認めれば裁判所へ通知しない(=過料は科されない)。
- 登記官が義務違反を知る典型場面: 相続人が別の不動産について遺言・遺産分割に基づく相続登記を申請した際、その添付書面から同じ相続人が他の不動産も取得していると判明した場合等。

### 遺贈の扱い
- 第76条の2第1項括弧書により、**相続人に対する遺贈**で所有権を取得した者も義務化の対象。
- 相続人への遺贈による所有権移転登記は、別の改正(不登法63条第3項、令和5年4月1日施行)で**登記権利者の単独申請**が可能になっている。
- 要確認: 相続人以外の第三者への遺贈が義務対象外かは公式ページに明示がなく、条文構造上の解釈。

## 2. 相続人申告登記(不動産登記法第76条の3、令和6年4月1日施行)

### 趣旨
3年以内の相続登記(遺産分割を要する本来の登記)が期限内に難しい場合に、義務を簡易に履行できる仕組み。

### 必要書類(遺産分割協議は不要)
- 申出書
- 申出人が被相続人の相続人であることが分かる戸籍証明書
- 申出人の住所を証する情報
- (代理の場合)委任状

法定相続人の範囲・法定相続分の確定は不要。被相続人の出生から死亡までの戸籍を網羅的に集める必要がなく、申出人自身が相続人と分かる戸籍だけでよい点が本来の相続登記との最大の差。

### 効果と限界
- 報告的登記であり、**持分(法定相続分)は登記されない**。権利関係を公示するものではない。
- 申出をした相続人については、相続登記の申請義務を履行したものとみなされる(過料を免れる)。
- 限界: 遺産分割に基づく相続登記の申請義務は履行できない。後日遺産分割が成立したら、その成立日から3年以内に別途、遺産分割の内容に応じた相続登記が必要(76条の3第4項)。

### 登録免許税
- 非課税。

### オンライン
- ブラウザ版「かんたん登記供託申請」でオンライン申出が可能。押印・電子署名不要。本登記である所有権移転登記は同サービスの対象外。

## 3. 氏名・住所等変更登記の義務化(令和8年4月1日施行)

2026-06-27時点で施行済み。相続登記とは別系統だが、相続後の名義人にも継続的に関わる。

### 根拠条文
- 申請義務: 不動産登記法第76条の5
- 職権による変更登記: 同第76条の6
- 過料: 同法第164条第2項

### 義務の内容
- 所有権の登記名義人は、氏名・名称または住所に変更があったときは、**変更日から2年以内**に変更登記を申請する義務。
- 正当な理由なく怠った場合: **5万円以下の過料**。
- 経過措置: 施行日前にすでに変更して未登記の場合も対象。期限は**令和10年(2028)3月31日まで**(または変更日から2年のいずれか遅い日)。

### 職権による変更登記(スマート変更登記)
- 登記官が住基ネットの情報を検索し、職権で変更登記を行う。令和8年4月1日から開始。
- **検索用情報の申出**(職権登記の前提): 申し出る情報は氏名/氏名の振り仮名(外国籍はローマ字表音)/住所/生年月日/メールアドレス。受付開始は令和7年4月21日。職権登記時、登記官がメールアドレス宛に確認メールを送り、本人確認を経て変更登記を実施。
- 自然人=検索用情報の申出+住基ネット連携。法人=検索用情報の対象外で、会社法人等番号を登記しておくことで商業・法人登記システムと連携。外国人=振り仮名はローマ字表音。
- 関連改正: 外国に住所を有する登記名義人は、別途国内連絡先を登記する制度(不登法73条の2、令和6年4月1日施行)がある。

## 4. 相続土地国庫帰属制度(令和5年4月27日施行)

相続した不要な土地の所有権を国に引き取ってもらう制度。相続登記とは別手続だが相続案内で言及されうる。

- 根拠: 相続土地国庫帰属法(令和3年法律25号)。
- 利用できる人: 相続または相続人に対する遺贈で土地を取得した者。売買等で取得した者は不可。共有地は共有者全員の共同申請。
- 却下事由(申請段階で直ちに却下): 建物がある土地/担保権・使用収益権が設定された土地/他人の利用が予定された土地/土壌汚染土地/境界不明・所有権の存否や範囲に争いがある土地。
- 不承認事由(審査後に不承認): 一定の崖/管理処分を阻害する有体物が地上にある/除去が必要な地下の有体物がある/隣接地所有者等との争訟を要する/その他過分な費用・労力を要する。
- 審査手数料: 土地一筆当たり14,000円(収入印紙、返還不可)。
- 負担金: 10年分の標準的な管理費用相当額。原則20万円。市街化区域・用途地域内の宅地や農地、森林等の一部は面積に応じて算定(20万円超もあり)。法務省提供のExcel計算シートあり。
- 納付期限: 負担金通知到達の翌日から30日以内。納付時に所有権が国へ移転。期限内不納付は承認が失効。

## 5. 所有不動産記録証明制度(令和8年2月2日施行)

被相続人名義の不動産の把握を容易にし、相続登記の登記漏れを防ぐ制度。2026-06-27時点で施行済み。**施行日は令和8年2月2日**(2月3日ではない)。

- 根拠条文: 不動産登記法第119条の2。
- 内容: 特定の登記名義人が所有権登記名義人として記録されている不動産を一覧化(リスト化)した証明書を交付。
- 請求できる者: 所有権登記名義人本人(法人含む)/その相続人その他の一般承継人(法人含む)/上記の代理人。
- 手数料(検索条件1件につき1通当たり): 書面請求1,600円/オンライン請求・郵送交付1,500円/オンライン請求・窓口交付1,470円。
- 請求方法: 全国いずれの法務局でも書面・オンライン・郵送で可。オンラインは申請用総合ソフトを使用。
- 限界: 登記記録の氏名・住所が一致するものを検索する制度のため、旧住所・旧氏名のままの不動産やデータ不一致があると捕捉漏れの可能性。完全性は保証されない。

## 6. 法定相続情報証明制度との位置づけ

平成29年(2017)5月開始の既存制度。詳細は[05](05-hitsuyou-shorui-koseki.md)。義務化との連携点として、申請書の添付情報欄に「登記原因証明情報(法定相続情報番号)」と記載すれば、法定相続情報一覧図の写しの添付を省略できる(令和6年4月1日以降)。
