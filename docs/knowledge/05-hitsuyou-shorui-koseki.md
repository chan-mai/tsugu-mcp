# 05 必要書類と戸籍収集

相続登記の添付書類と、その中核である戸籍の集め方。

## 0. 添付情報の4分類

| 分類 | 役割 | 主な書類 |
|---|---|---|
| 登記原因証明情報 | 相続人を確定する | 被相続人の出生〜死亡の連続戸籍、相続人全員の現在戸籍、(協議時)遺産分割協議書+印鑑証明書、(遺言時)遺言書 |
| 住所証明情報 | 新名義人の住所を証明 | 相続人(新所有者)の住民票の写し |
| 同一性証明 | 登記簿上の名義人=被相続人を証明 | 被相続人の住民票の除票または戸籍の附票 |
| 課税価格の根拠 | 登録免許税の計算 | 固定資産課税明細書/評価証明書/名寄帳 |

法定相続情報一覧図の写しを使えば、登記原因証明情報のうち戸籍束の提出を省略でき、住所記載があれば住所証明情報も兼ねられる(後述3)。

## 1. 登記原因証明情報としての戸籍

### 1-1. 被相続人の出生〜死亡の連続戸籍が必要な理由
日本国民は出生で父母の戸籍に入り、婚姻で新たな夫婦の戸籍に入るというように、出生から死亡まで複数の戸籍に記録される。法定相続人を特定するため、ある一時点の戸籍だけでは過去の婚姻歴・認知した子・養子など別戸籍の相続人を見落とす。よって全戸籍を連続して集めて初めて相続人を確定できる。

戸籍が新しく作られる(前の戸籍が閉じる)主因:
- 身分行為: 婚姻・離婚、養子縁組・離縁、家督相続(旧民法)
- 戸籍手続: 転籍(本籍地の変更)、分籍
- 法令改製: 昭和改製(昭和32年法務省令27号、家制度廃止)、平成改製(平成6年法務省令51号、コンピュータ化)

改製では新様式の戸籍が作り直されるが、改製時点で既に除籍されていた人(婚姻・離婚・死亡で抜けた子など)は新戸籍に転記されない。よって改製後の戸籍だけでは改製前の相続人を確認できず、改製原戸籍まで遡る必要がある。これが出生まで遡る中心的理由。

### 1-2. 相続人全員の現在戸籍(死亡日以降に取得)
相続人の戸籍は「被相続人が死亡した日以後の証明日のもの」。相続は被相続人の死亡時点で生存している相続人に発生するため、相続人が相続開始時に生存していたことを死亡日以降の証明日の戸籍で確認する。

### 1-3. 相続パターン別の戸籍範囲
相続人の順位は配偶者(常に相続人)+第1順位=子(直系卑属)/第2順位=直系尊属/第3順位=兄弟姉妹(詳細は[07](07-souzokunin-soutokubun.md))。どのパターンでも共通の土台は「被相続人の出生〜死亡の連続戸籍」+「相続人全員の現在戸籍(死亡日以降)」。

| 相続パターン | 被相続人について | 追加で必要な戸籍と趣旨 |
|---|---|---|
| 配偶者 | 出生〜死亡の連続戸籍一式 | 配偶者は通常死亡時戸籍に同籍。出生〜死亡戸籍で他の同順位相続人も確認 |
| 第1順位: 子 | 出生〜死亡の連続戸籍一式 | 前婚の子・認知した子・養子を含む全ての子を確定 |
| 代襲(子先死亡→孫) | 同上 | **先に死亡した子の出生〜死亡の連続戸籍**を追加。孫を漏れなく確定 |
| 第2順位: 直系尊属 | 出生〜死亡の連続戸籍一式(子・孫が不存在の証明) | 父母死亡で祖父母が相続する場合は父母の死亡が分かる戸籍。直系尊属に代襲なし |
| 第3順位: 兄弟姉妹 | 出生〜死亡の連続戸籍一式(子・孫・直系尊属が不存在の証明) | **被相続人の父・母それぞれの出生〜死亡の連続戸籍**を追加。両親の全ての子=兄弟姉妹を確定。直系尊属全員死亡の確認も兼ねる |
| 代襲(兄弟姉妹先死亡→甥姪) | 同上 | 先に死亡した兄弟姉妹の出生〜死亡の連続戸籍を追加。甥姪を確定。一代限り(再代襲なし) |

第2・第3順位の戸籍範囲は法務局の必要書類一覧の注記を専門家解説が具体化したもの(法令根拠は民法887〜889条)。

### 1-4. 戸籍の種類の用語
| 紙(コンピュータ化前) | コンピュータ化後 | 意味 |
|---|---|---|
| 戸籍謄本 | 戸籍全部事項証明書 | 戸籍記載の全員の証明 |
| 戸籍抄本 | 戸籍個人事項証明書 | 戸籍記載の一部の個人の証明 |
| 除籍謄本 | 除籍全部事項証明書 | 記載者全員が婚姻・死亡・転籍等で除かれた戸籍 |
| 改製原戸籍謄本 | (紙のまま) | 戸籍の制度改正で作り変える前の戸籍 |

改製原戸籍の読みは「かいせいげんこせき」。「現戸籍」と紛らわしく実務では「はらこせき」と通称。

## 2. 戸籍の広域交付(令和6年3月1日施行)

戸籍法改正(令和元年法律17号)。

- どこでも: 本籍地が遠くても住所地や勤務先の最寄りの市区町村窓口で請求できる。
- まとめて: ほしい戸籍の本籍地が全国各地にあっても1か所の窓口で一括請求できる。
- 利用できる人: 窓口に来た本人から見て、本人・配偶者・直系尊属・直系卑属の戸籍に限定。

利用できないケース(注意すべき「3つの穴」を含む):

| 制約 | 内容 |
|---|---|
| 代理人・第三者 | 不可。司法書士等の職務上請求も対象外 |
| 郵送請求 | 不可。本人が窓口へ出向く |
| 証明書の種類 | 一部事項証明書・個人事項証明書(抄本)は不可、全部事項証明書のみ |
| **兄弟姉妹(傍系)の戸籍** | 対象外 |
| **電子化されていない戸籍** | 一部の改製原戸籍等は対象外(本籍地へ請求) |
| **戸籍の附票** | 対象外(本籍地でのみ取得) |

本人確認: 顔写真付きの官公署発行身分証明書(マイナンバーカード・運転免許証・パスポート等)。相続では出生〜死亡の一連戸籍を最寄り1か所で取得できるのが最大のメリットだが、第3順位(兄弟姉妹)案内では3つの穴を必ず注意喚起。

## 3. 法定相続情報証明制度(平成29年5月29日開始)

戸除籍謄本の束の代わりに、登記官の認証文付き「法定相続情報一覧図の写し」を無料で必要通数交付する制度。相続登記・預貯金払戻し・相続税申告・年金手続等に横断利用できる。

### 法定相続情報一覧図
被相続人の氏名・最後の住所・生年月日・死亡年月日(任意で最後の本籍)、各相続人の氏名・生年月日・続柄を記載。相続人の住所記載は任意。申出人が自分で作成(法務局が様式・記載例を公開)。

### 申出の必要書類
必須:
1. 被相続人の戸除籍謄本(出生〜死亡の連続一式)
2. 被相続人の住民票の除票
3. 相続人全員の現在の戸籍謄抄本
4. 申出人の本人確認書類(運転免許証コピー等、原本相違ない旨+記名)

場合により:
5. 各相続人の住民票の写し(一覧図に相続人の住所を記載する場合)
6. 委任状(代理人が手続する場合)
7. 被相続人の戸籍の附票(住民票の除票が取得できない場合)

申出人になれる人: 被相続人の相続人(数次相続ならその相続人)。代理人=法定代理人、民法上の親族、資格者代理人(弁護士・司法書士・土地家屋調査士・税理士・社会保険労務士・弁理士・海事代理士・行政書士)。

申出ができる登記所(いずれか): 被相続人の本籍地/最後の住所地/申出人の住所地/被相続人名義の不動産の所在地。

### 交付・手数料
手数料無料。必要通数を何枚でも交付。提出した戸除籍原本は確認後に返却。一覧図は申出日の翌年から5年間保存され、その間は当初の申出人が再交付を受けられる。

### 相続登記での活用
- 戸除籍束の代わりに一覧図の写しを添付できる。**法定相続情報番号**を申請書に記載する方法でも、被相続人の戸除籍と相続人の戸籍の添付に代えられる。
- 複数管轄に不動産があっても各法務局へ写しを提出すればよく原本束を重複提出しなくてよい。
- 一覧図の写しに相続人の現住所が記載されていれば住所証明情報の添付に代えられる。

相続関係説明図との違いは[06](06-fuzui-shorui.md)。

## 4. 住所・評価関係の書類

### 4-1. 被相続人の同一性証明(住民票の除票または戸籍の附票)
「被相続人の登記上の住所」が「戸籍の本籍」と異なる場合、登記上の所有者と戸籍上の被相続人が同一人であることを証明するため、次のいずれかを添付。
- 住民票の写し(本籍及び登記上の住所が記載されたもの)
- 住民票の除票の写し(同上)
- 戸籍の附票の写し(戸籍の表示及び登記上の住所が記載されたもの)

役割: 登記記録上の住所と戸籍上の本籍をつなぐ。登記上の住所と本籍が一致していれば不要。有効期限なし。

エスカレーション(公式手続案内に明文なし=要確認): 除票・附票も廃棄され住所変遷を登記簿上の住所まで遡れない場合、不在籍・不在住証明書+固定資産税証明の組合せ、被相続人名義の権利証(登記済証/登記識別情報)提供、相続人全員の上申書(実印+印鑑証明書)などで補う。

### 4-2. 相続人(新名義人)の住所証明情報
相続した人全員の住民票の写し(原本)。**マイナンバー(個人番号)が記載されていないもの**。住民票コードを申請書に記載すれば添付を省略できる。

### 4-3. 課税価格の根拠(固定資産評価)
課税価格は市区町村の固定資産課税台帳の価格。確認手段: 固定資産課税明細書(毎年4月頃に納税通知書に同封、申請日の属する年度=最新年度のもの)/固定資産評価証明書・課税証明書(市区町村で取得)/名寄帳。公衆用道路など課税台帳に価格がない不動産は登記所が認定した価額。
- 誤りやすい点: 使うのは「価格」「評価額」であって「固定資産税課税標準額」ではない。
- 取得先: 市区町村役場。東京23区は都税事務所(都が課税)。相続人が取得するには被相続人の死亡が分かる書類と相続人の現在戸籍、または法定相続情報一覧図が必要。

### 4-4. 印鑑証明書
- 遺産分割協議による相続登記: 遺産分割協議書に相続人全員が実印を押し、その印鑑証明書を各1通添付。
- **有効期限なし**: 相続登記(遺産分割協議書添付)の印鑑証明書には作成後3か月以内などの制約はない。売買等で登記義務者の印鑑証明書が3か月以内を要するのと対照的。
- 金融機関の預貯金相続では発行後3か月や6か月以内を求めるのが一般的(法律上の期限ではなく各行の運用)。相続登記側に期限がない点と異なる。
- 不要なケース: 法定相続分どおりの相続登記、遺言による相続登記(協議書を作らない)。

## 5. 原本還付

申請人(または代理人)が原本のコピーを作り、コピーに「原本に相違ない」旨を付記して署名・押印すれば原本の返還を請求できる。登記官は原本で審査後、原本とコピーを照合して原本を返還。別途の還付請求書は不要。複数枚はつづり目に契印。

書類別の分岐:
- 戸籍謄本・除籍謄本 → **相続関係説明図を提出すればコピー不要で原本還付**(戸籍束の返却が主目的)。
- 住民票・遺産分割協議書・印鑑証明書など → コピー(原本相違ない旨+申請人の記名押印)を付けて原本還付請求。
- 委任状・登記原因証明情報・印鑑証明書等で当該申請のためだけに作成したもの → 還付不可。

法定相続情報一覧図の写しとの関係: 相続関係説明図は戸籍原本の還付を受けるための図(戸籍束は提出する)。法定相続情報一覧図の写しは認証付きで戸籍束の提出自体を不要にする制度。後者を使えば登記・銀行等を並行しやすい。

## 6. 取得方法の実務

### コンビニ交付(マイナンバーカード)
取得できる証明書: 住民票の写し・住民票記載事項証明書・印鑑登録証明書・各種税証明書・戸籍証明書(全部/個人事項証明)・戸籍の附票の写し。マイナンバーカード(またはスマホ用電子証明書)+利用者証明用電子証明書の暗証番号(4桁)が必要。
- **本籍地と住所地が異なる場合**: 戸籍証明書・戸籍の附票をコンビニ交付するには、事前に本籍地の市区町村へ「戸籍証明書のコンビニ交付利用登録申請」が必要。登録完了まで概ね3〜5営業日。**即日取得できない点は誤解が多い**。
- 利用時間: 多くで概ね6:30〜23:00(年末年始・メンテ除く、自治体差あり)。

### 郵送請求
広域交付・コンビニ交付が使えない場合(兄弟姉妹の戸籍、電子化前の戸籍、戸籍の附票など)は本籍地へ郵送請求。必要なもの: 請求書/手数料(定額小為替、おつりが出ないよう同額、有効期間は発行日から6か月)/返信用封筒(切手貼付)/本人確認書類のコピー。直系以外は関係を示す疎明資料が追加で必要なことがある。受取まで2週間程度かかる場合あり。

### 手数料の目安
| 証明書 | 窓口手数料(1通) |
|---|---|
| 戸籍全部/個人事項証明書(謄抄本) | 450円(全国共通) |
| 除籍全部事項証明書(除籍謄本) | 750円(全国共通) |
| 改製原戸籍謄本 | 750円(全国共通) |
| 戸籍の附票の写し | 約300円(自治体差) |
| 住民票の写し | 約300円(自治体差) |
| 印鑑登録証明書 | 約300円(自治体差) |

戸籍系(450円/750円)は手数料令により全国共通。住民票・印鑑証明・附票は自治体差(250〜400円程度)。コンビニ交付は窓口より100円程度安い自治体が多い。
