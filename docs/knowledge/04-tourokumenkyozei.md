# 04 登録免許税

相続登記の登録免許税の計算・端数処理・免税措置・納付。

> 最新の注意: 免税措置の根拠条文は、令和8年度税制改正により令和8年4月1日から「租税特別措置法第84条の2の3」→**「第84条の2の2」に条数変更**された。2026-06-27時点で正しい条文は**第84条の2の2**。適用期限は**令和9年(2027)3月31日**まで。旧条文を申請書に書くと免税を受けられない恐れがある。

## 1. 基本計算

### 計算式
```
登録免許税額 = 課税標準 × 税率
```
相続による所有権移転登記の課税標準は「不動産の価額」。

### 課税標準(不動産の価額)
- 市町村が管理する**固定資産課税台帳に登録された価格(評価額)**。毎年市町村から届く固定資産課税明細書の「価格」または「評価額」。
- 注意: 明細書の**「固定資産税課税標準額」ではない**。住宅用地特例等で減額された後の課税標準額と、登録免許税の課税標準になる価格(評価額)は別物。必ず「価格」「評価額」欄を使う。
- 確認方法: 固定資産課税明細書/固定資産評価証明書/名寄帳(課税台帳の写し)。
- **1,000円未満切り捨て**。価格が1,000円未満なら1,000円。
- **複数不動産を同一申請書で申請するときは、各不動産の価格を合計してから1,000円未満を切り捨てる**(各物件ごとに切り捨ててから足すのではない)。

### 税率(相続)
- 相続・法人の合併による移転、共有物分割による移転(持分相当部分に限る): **1000分の4(0.4%)**。
- 贈与・財産分与などその他の原因: 1000分の20。

### 税額の端数処理
- 税額に100円未満の端数があれば**切り捨て**。
- 計算した税額が**1,000円未満なら1,000円**(最低税額)。
- 土地と建物を同一申請書で申請し税率が異なる場合は、税率ごとに課税標準×税率を計算し、その合計額から100円未満を切り捨てる。

### 評価額のない土地(公衆用道路・私道など)
- 固定資産課税台帳に価格がない(評価額0円・非課税地)土地でも登録免許税は課税される。課税標準は**登記官が認定した価額**。
- 実務上の認定: **近傍宅地の1㎡あたり評価額 × 100分の30** を私道の単価とする(法務局長通達による認定基準)。
  ```
  課税価格 = 近傍宅地の1㎡単価 × 私道の地積(㎡) × 30/100 × 持分割合
  ```
- 近傍宅地単価は、評価証明書の記載/本地の評価額÷地積/管轄登記所の指定で得る。**事前照会推奨**(登記所により扱いが分かれる)。

## 2. 免税措置(土地のみ対象、建物は対象外。期限=令和9年3月31日)

両措置とも令和7年度税制改正で2年延長され適用期限は令和9年(2027)3月31日まで。令和8年度税制改正で条数が第84条の2の2に変更(令和8年4月1日施行)。**記載がないと免税を受けられない**(自動適用されない)。

### 措置1: 数次相続の中間者(死亡した相続人)の免税 — 第84条の2の2第1項
- 相続(相続人への遺贈を含む)で土地を取得した個人が、その相続による所有権移転登記を受ける前に死亡した場合、その死亡した個人を登記名義人とするために受ける登記は非課税。
- A→B→Cと相続が連続したとき、中間のBが登記前に死亡なら**A→B(1次相続)への移転登記**が免税。**B→C(2次相続)への移転登記は免税対象外**(通常の0.4%)。
- 対象は土地のみ。
- 申請書記載: `登録免許税　租税特別措置法第84条の2の2第1項により非課税`

### 措置2: 価額100万円以下の土地の免税 — 第84条の2の2第2項
- 土地について(a)相続による所有権移転登記、または(b)表題部所有者の相続人が受ける所有権**保存**登記で、課税標準となる土地の価額が**100万円以下**のとき非課税。
- 上限額の変遷: 当初(平成30年)は市街化区域外で法務大臣指定の評価額10万円以下 → 令和4年度改正で全国の土地に拡充・上限10万円→100万円 → 令和7年度改正で期限を令和9年3月31日まで2年延長。
- 100万円判定は**課税標準となる価額**(持分移転なら価額×持分)。複数筆は各筆ごとに判定し、該当する筆のみ免税。
- 申請書記載: `登録免許税　租税特別措置法第84条の2の2第2項により非課税`

混在時: 1件の申請に免税対象の土地と課税対象の不動産が混在する場合、課税分のみ課税標準・税額を記載し、免税分は条項を併記する。

## 3. 関連税率(比較用、2026-06-27時点)

| 登記の種類 | 課税標準 | 本則税率 | 軽減/特例 |
|---|---|---|---|
| 所有権保存 | 不動産の価額 | 0.4% | 住宅用家屋0.15%(R9.3.31まで, 72条の2) |
| 所有権移転(相続・合併・共有物分割) | 不動産の価額 | 0.4% | 上記免税措置 |
| 所有権移転(売買・土地) | 不動産の価額 | 2.0% | **1.5%(R11.3.31まで, 72条1項)** |
| 所有権移転(売買・建物) | 不動産の価額 | 2.0% | 住宅用家屋0.3%(R9.3.31まで, 73条) |
| 所有権移転(贈与・財産分与等) | 不動産の価額 | 2.0% | — |
| 抵当権設定 | 債権金額(根抵当は極度額) | 0.4% | 住宅資金0.1%(R9.3.31まで, 75条) |
| 配偶者居住権設定 | 不動産の価額 | 0.2% | — |
| 抹消・住所氏名変更等 | 不動産の個数 | 1個1,000円(同一申請20個以上は2万円) | 住居表示実施に伴う住所変更は非課税 |

土地売買の軽減税率1.5%だけ期限が異なり**令和11年(2029)3月31日まで**。住宅用家屋・住宅ローン抵当権の軽減はR9.3.31まで。

## 4. 納付方法

- 原則: 現金を国(税務署等の金融機関窓口)に納付し、領収証書を申請書に貼り付けて提出。
- 収入印紙: 税額が3万円以下の場合その他特別の場合は収入印紙で納付できる(実務では3万円超でも印紙納付が広く行われている)。
- オンライン納付(電子納付): オンライン申請・QRコード付き書面申請ではペイジー等で電子納付できる。
- 貼付台紙と契印: 領収証書・収入印紙は別葉の白紙(収入印紙貼付台紙)に貼り、申請書とつづって申請書と台紙の間に契印。**収入印紙には割印・消印をしない**。

## 5. 計算例

### 例A: 土地+建物を相続(同一申請書)
土地の評価額12,345,678円、建物の評価額5,432,100円(ともに相続0.4%)
```
課税標準: 12,345,678 + 5,432,100 = 17,777,778 → 17,777,000円(合計後1000円未満切捨)
税額:     17,777,000 × 4/1000 = 71,108円 → 71,100円(100円未満切捨)
→ 登録免許税 71,100円
```

### 例B: 複数の土地を相続(同一申請書)
土地1=4,478,400円、土地2=3,489,100円(相続)
```
課税標準: 4,478,400 + 3,489,100 = 7,967,500 → 7,967,000円
税額:     7,967,000 × 4/1000 = 31,868円 → 31,800円
→ 登録免許税 31,800円
```

### 例C: 私道(公衆用道路・評価額0円)単独
近傍宅地単価125,000円/㎡、私道地積50㎡、持分1/7
```
認定価額 = 125,000 × 50 × 30/100 × 1/7 = 267,857円 → 267,000円(1000円未満切捨)
税額     = 267,000 × 4/1000 = 1,068円 → 1,000円(1000円未満は最低額1000円)
※ 同一申請の他の不動産があれば合算してから切捨て
```

### 例D: 100万円以下の土地(免税・措置2)
山林の評価額850,000円(相続、100万円以下)
```
→ 課税価格・税額は記載せず、登録免許税欄に
  「租税特別措置法第84条の2の2第2項により非課税」と記載
→ 登録免許税 0円
(同一申請に課税対象の宅地500万円があれば、宅地分のみ 5,000,000 × 4/1000 = 20,000円 を課税)
```

### 例E: 最低税額
評価額90,000円の土地(相続、免税対象外を想定)
```
税額: 90,000 × 4/1000 = 360円 → 1,000円未満なので 1,000円
→ 登録免許税 1,000円
```

## 6. 計算手順(端数処理の順序)

```
1. 各不動産の課税価格を算出(評価額×持分。評価額0円の私道は 近傍宅地単価×地積×0.3×持分 で認定)
2. 同一申請内で税率が同じ不動産は課税標準を合算 → 合計から1,000円未満切り捨て
3. 税率を乗じる(相続=0.4%)。税率が異なる不動産が混在する場合は税率ごとに2→3を行い税額を合算
4. 税額(合算後)の100円未満を切り捨て
5. 結果が1,000円未満なら1,000円
6. 免税該当物件(措置1・2)は課税計算から除外し、申請書には根拠条項(第84条の2の2第1項/第2項)を出力
```

適用期限は令和9年(2027)3月31日(土地売買の軽減1.5%は令和11年3月31日)。期限後は要再確認。
